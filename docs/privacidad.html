Política de privacidad

En este Sitio no se requiere registro y el cálculo se realiza en tu navegador. El pago se procesa mediante Stripe. El Titular no almacena de forma centralizada los datos introducidos para el cálculo o el documento, salvo que el usuario los remita voluntariamente por email para soporte.

Datos que podemos tratar

  • Soporte por email: si nos escribes, trataremos tu email y el contenido del mensaje para atenderte.
  • Pagos: Stripe trata los datos necesarios para procesar el pago. El Titular recibe confirmación del pago y datos mínimos para la gestión (p. ej. email e identificadores de la transacción).
  • Estado de desbloqueo: el Sitio puede usar almacenamiento local del navegador para recordar el desbloqueo tras una compra en el mismo dispositivo.

Finalidades

  • Atender solicitudes de soporte.
  • Gestionar pagos, devoluciones, prevención del fraude y obligaciones contables.

Base jurídica

  • Ejecución de un contrato (compra del documento o suscripción).
  • Cumplimiento de obligaciones legales (contables/fiscales).
  • Interés legítimo (soporte y prevención del fraude).

Conservación

Las comunicaciones de soporte se conservan el tiempo necesario para atender la solicitud y, en su caso, para la defensa ante reclamaciones. Los datos de facturación y transacciones se conservan según obligaciones legales.
